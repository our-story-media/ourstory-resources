Install Software onto Devices

A custom application will need to be installed on each Android device to use it with Indaba. There are two options to install the App on your devices.

You will be prompted to enable Unknown Sources when installing this application, this is normal for applications not from the Google Play Store and is OK to accept in this instance.

For Online:

  • Download and manually install from GitHub Release.

For Titan:

  • Download and manually install from GitHub Releases.
