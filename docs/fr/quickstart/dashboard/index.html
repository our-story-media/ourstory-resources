Tableau de bord

Le tableau de bord est l'endroit où vous configurez tous les paramètres, flux de travail et contenu en ce qui concerne Indaba. Seuls les administrateurs Shoot ont accès au tableau de bord, soit en utilisant Titan Edition directement sur un ordinateur, soit via une connexion Google en utilisant Online Edition.

Commencer

Visitez le tableau de bord (en ligne ou en démarrant l'application Titan sur votre ordinateur) et lancez une nouvelle prise de vue .

  1. Créez un nouveau tournage pour votre déploiement, en veillant à remplir la plage de dates et le nom.
  1. Les paramètres par défaut doivent être OK pour la plupart des déploiements, mais veuillez vérifier avant d'entrer dans le champ.
  1. Téléchargez des images adaptées au contexte pour chaque thème dans l'écran Modèle .

Les paramètres d'accès et d'autorisations par défaut pour l'édition en ligne autorisent uniquement la contribution à votre tournage à l'aide d'un code à 4 chiffres. Ne prenez pas ce code à remettre aux participants. Contribution dans Titan Edition est ouvert à tous.

** Prenez note du nom ou du code que vous souhaitez partager avec les animateurs ou les participants. \
Il s'agit de la configuration minimale requise pour déployer Indaba. **

Configuration optionnelle

Plusieurs éléments peuvent être configurés avant d' entrer dans un déploiement:

Mots clés

Les balises sont les étiquettes que les participants utilisent pour sélectionner et regrouper les vidéos en histoires. Le nom, la couleur et la langue de ces balises peuvent être ajustés pour s'aligner sur votre cas d'utilisation spécifique. La configuration par défaut comprend des balises basées sur les critères du CAD de l'OCDE pour le suivi et l'évaluation.

Modèle

Pour mieux s'aligner sur votre contexte de déploiement, vous pouvez ajuster le modèle - l'ensemble de plans et de descriptions suggérés pour les participants. Il existe une infinité d'options pour créer ces modèles, ou vous pouvez utiliser l'un des modèles existants dans Indaba. Nous fournissons également une grande archive de suggestions de plans pour créer le vôtre.
